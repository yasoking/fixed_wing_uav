# Fixed-Wing UAV Graduation Project

## Overview
This project presents the design, analysis, and manufacturing of a fixed-wing unmanned aerial vehicle (UAV). The aircraft was developed as a graduation project in Mechatronics Engineering.

## Key Features
- Fixed-wing aircraft configuration
- Carbon fiber structure
- Aerodynamic analysis
- Stability and performance calculations
- CAD design and manufacturing
- Flight-ready prototype

## Tools Used
- SolidWorks
- MATLAB
- CNC Manufacturing
- Carbon Fiber Composite Fabrication
